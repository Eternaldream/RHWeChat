# RHWeChat
### 无需越狱逆向微信，个人想到的一些逆向微信的好的点子实现！
* 自动抢红包
 
### 使用到的工具：
* [MonkeyDev](https://github.com/AloneMonkey/MonkeyDev)
* [Reveal4](https://revealapp.com/)
* [cycript](http://www.cycript.org/)

### 插件使用方法：
* 点击设置进去，打开自动抢红包开关，默认关闭
